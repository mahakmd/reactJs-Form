# React + Vite

sing up form using react js and formik and yup!



Setup
To clone the repository on your computer
```
$ https://github.com/mahakmd/reactJs-Form.git
```
```
$ cd reactJs-Form
```
To install dependencies,
```
$ npm i
```
To run the project itself,
```
$ npm run dev
```
